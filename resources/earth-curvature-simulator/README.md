# Earth Curvature & Horizon Simulator

> A **member-generated resource**, not an experiment: a to-scale, interactive
> model of curvature, horizon dip, hidden height, and atmospheric refraction.
> It makes the predicted geometry visible and checkable — it does **not** measure
> anything, and it asserts nothing about the world. Only a measured run does that.

▶ **[Open the simulator](simulator.html)**
&nbsp;·&nbsp; [live](https://replicationbench.github.io/RB/resources/earth-curvature-simulator/simulator.html)

Supports the experiment it was built alongside:
**[Water-Surface Shape by Laser](../../experiments/water-surface-shape-laser/)**.

## What it does

- **Side, Observer, and Laser-Test** views at true 1:1 scale (no vertical
  exaggeration unless explicitly labelled).
- Selectable **atmospheric refraction** (effective-radius *k*), a WGS84 oblate
  option, and famous long-distance observations as presets.
- **Hidden-height, horizon-distance/dip, and drop** math, with shareable
  permalinks and PNG/CSV export.
- A **Laser Test** view with level-at-origin, aim-by-sight (the FECORE error made
  interactive), and beam-vs-target readouts.

A single dependency-free HTML file. Resources like this are community
contributions that help readers reason about an experiment; they live under
`resources/` precisely because they are not themselves measurements.
